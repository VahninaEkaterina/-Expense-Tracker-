# Expense Tracker

**Автор:** Екатерина Вахнина
**Описание:** Приложение для отслеживания личных расходов с графическим интерфейсом (Tkinter), фильтрацией по категории и дате (периоду), подсчётом общей суммы и сохранением данных в формате JSON.

## Как использовать

1.  **Запуск:**
    Откройте терминал в папке проекта и выполните команду:
    ```bash
    python app.py
    ```
2.  **Добавление расхода:**
    *   Введите сумму (например: `150.5`).
    *   Выберите категорию из списка (Еда/Транспорт/...).
    *   Введите дату в формате ГГГГ-ММ-ДД (например: `2026-04-30`).
    *   Нажмите кнопку **«Добавить расход»**.
3.  **Просмотр истории:**
    Все добавленные расходы отображаются в таблице выше.
4.  **Фильтрация:**
    *   Выберите категорию в поле «Категория» или оставьте «Все».
    *   Введите даты начала и окончания периода в поля «Дата с» и «Дата по».
    *   Нажмите **«Применить фильтр»**. Таблица и сумма обновятся.
5.  **Сумма расходов:**
    Текущая сумма расходов (с учётом фильтра) отображается под таблицей.
6.  **Сохранение:**
    Данные сохраняются автоматически в файл `expenses.json`.
7.  **Загрузка:**
    При запуске приложения история загружается из `expenses.json`.

## Примеры использования (тесты)
*   **Ошибка суммы:** Введите `-10` или `abc` и нажмите «Добавить расход» — появится окно с ошибкой.
*   **Ошибка даты:** Введите `30-04-2026` или `апрель` и нажмите «Добавить расход» — появится окно с ошибкой.
*   **Фильтр по категории:** Выберите «Еда» и нажмите «Применить фильтр» — в таблице останутся только траты на еду.
*   **Фильтр по периоду:** Введите даты `2026-04-28` и `2026-04-30` — отобразятся расходы только за эти дни.
*   **Комбинированный фильтр:** Выберите категорию «Транспорт» и укажите период — отобразятся траты на транспорт за выбранное время.
